自己回帰モデルによる株価予測

  • 自己回帰モデル \[X_t=c+\sum_{i=1}^{p}\phi_iX_{t-i}+\epsilon_t\] （\(c\)は定数項，\(\epsilon_t\)は白色雑音，\(\phi_i\)はモデルのパラメータ）
  • このモデルを用いて1期（1日）先予測を10期分行い真値と比較
df <- read.csv("../data-raw/nm2018.csv",header=T,row.names=1)
adjclose_data<-as.ts(df["adjustedClose"])
j <- 1
ar_predict_result <- matrix()
for(i in 211:220){
     training_data <- adjclose_data[1:i]
     ar_data_adjclose <- ar(training_data)
     ar_predict_adjclose <- predict(ar_data_adjclose, ahead=1)
     ar_predict_result[j] <- ar_predict_adjclose$pred[1]
     j <- j+1
}
ts.plot(as.ts(ar_predict_result),xlim=c(0,10),ylim=c(950,1040), ann=FALSE)
par(new=T)
ts.plot(as.ts(adjclose_data[212:221]),xlim=c(0,10),ylim=c(950,1040),lty="dashed",ann=F)
legend("topright", lty=1:2, legend=c("predict", "true value"))
